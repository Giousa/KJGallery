## KJGallery简介
一个支持Gif图片以及普通图片预览，支持双击缩放，单机退出，同时可以选择使用jni的形式去高效加载gif或者更更精简的(仅2个类)gif控件     
详细介绍请查看博客：http://www.kymjs.com/code/2015/10/18/01/
